# MacroScope Public v4.0

一个不要求访问者填写 Token 的公开宏观市场网站。网站由 GitHub Pages 托管，GitHub Actions 定时抓取公开数据、保留历史缓存并重新生成静态网页。

## 核心变化

- 不再依赖本地 Streamlit 服务；网站部署后，任何人都可直接打开网址。
- 不需要访问者安装 Python、Docker或填写数据 Token。
- 数据抓取与网页访问分离：GitHub Actions 负责更新，GitHub Pages 只负责展示。
- 每个数据集均有独立状态与备用源；单个来源失败时，继续展示上次成功缓存。
- 数据健康页会明确显示成功、部分成功、失败或使用旧缓存，不用空白页面掩盖错误。

## 三个主要看板

### 1. 宏观流动性与价格

| 指标 | 展示单位 | 口径 |
|---|---:|---|
| M1、M2余额 | 万亿元 | 原始亿元 ÷ 10,000 |
| M1、M2同比 | % | 同比增速 |
| M1−M2剪刀差 | 百分点 | M1同比 − M2同比 |
| M1+M2机械合计 | 万亿元 | 仅按需求展示；M1已经包含在M2中，因此存在重复计算 |
| 社融当月增量 | 万亿元 | 社会融资规模增量 |
| 社融存量 | 万亿元 | 中国人民银行官方存量统计表 |
| 社融存量同比 | % | 标准“社会融资增速”口径 |
| 制造业/非制造业PMI | 指数点 | 50为荣枯线 |
| CPI同比/环比 | % | 全国居民消费价格指数 |

### 2. 中美科技龙头

- 科创50：`000688.SH`
- 中证科技龙头指数：`931087.CSI`，网页简称“中国科技龙头”
- 美股龙头：AAPL、MSFT、NVDA、AMZN、GOOGL、META、TSLA
- 展示最新收盘、当日涨跌、年初至今、近一年和标准化走势

### 3. 历史估值与交易拥挤度

估值覆盖：

- 上证综指
- 深证成指
- 创业板指
- 科创50
- 上证50
- 沪深300
- 中证500

展示当前 PE TTM、历史平均、中位数、25%/75%分位及当前历史分位。不同公开源不一定都提供 PB；网页不会使用虚构值填补。

交易拥挤度定义：

```text
成交额排名前5%的沪深A股成交额合计 ÷ 沪深A股总成交额 × 100%
```

“前5%”按股票数量计算。成交额统一换算为万亿元。

## 数据源与容错

主要公开源包括：中国人民银行统计表、东方财富公开行情、中证指数公开数据、AKShare适配器、Yahoo Finance/yfinance 和 Stooq。

程序采用以下策略减少缺失：

1. 同一指标配置多个来源或接口。
2. 每个指标独立更新，一个模块失败不阻断其他模块。
3. 新数据与历史CSV按日期和代码合并，不重复覆盖完整历史。
4. 抓取失败时保留上次成功缓存，并在“数据口径与运行状态”页标记。
5. 社融存量和同比直接读取中国人民银行年度 Excel 统计表；社融12个月累计增量同比只作为辅助备用口径。

公开网页可能改版、限频或延迟发布，因此不能保证任何第三方免密来源永远稳定。程序的目标是“可追踪地降级”，而不是静默展示空数据或错误数据。

## 一键部署成公开网站

完整步骤见：[`docs/DEPLOY_GITHUB_PAGES.md`](docs/DEPLOY_GITHUB_PAGES.md)

部署后网址通常为：

```text
https://你的GitHub用户名.github.io/仓库名/
```

默认自动更新：

- 北京时间每天 08:15：宏观、指数、美股、估值和缓存检查
- 北京时间工作日 16:40：收盘后行情与交易拥挤度更新
- GitHub Actions 页面也可手动点击 `Run workflow`

## 本地检查

Mac：右键打开 `START_LOCAL_MAC.command`。

Windows：双击 `START_LOCAL_WINDOWS.bat`。

本地地址：

```text
http://localhost:8000
```

本地模式不是公开网址，只用于检查网页。公开分享请部署到 GitHub Pages。

## 项目结构

```text
.github/workflows/update-and-deploy.yml  自动抓取与部署
config/settings.yaml                    指标和代码配置
src/providers.py                        数据源与备用源
src/pipeline.py                         缓存、合并与状态
scripts/update_data.py                  抓取数据
scripts/build_site.py                   生成 public/index.html
scripts/generate_demo_data.py           明确标注的界面演示数据
public/                                 GitHub Pages发布目录
data/                                   定时任务保留的CSV缓存
```
